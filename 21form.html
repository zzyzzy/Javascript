자바스크립트 DOM form

form 다루기 - button

폼의 각 요소를 다룰때 querySelector나 getElementById 대신 폼이름.name속성으로도 사용 가능함

폼에 입력한 값들을 서버로 전송하려면 버튼 객체의 submit 함수 호출

MIME : 클라이언트의 다양한 요청의 종류를 정의한 것 : 메세지타입/서브타입
웹 서버는 일반적으로 텍스트형식의 파일(text/plain)만 다룰수 있음
하지만, 이진파일(application/octet-stream)이나 JSP/PHP등의 파일등은 웹서버가 다룰 수 없기 때문에 그것을 처리할수 있는 추가적인 프로그램이 필요할 수 있음

form 다루기 - submit

submit 버튼에 대한 이벤트 처리는
1. form태그에 onsubmit 속성 정의하고
2. return 폼체크함수명으로 이벤트핸들러 등록함
3. 폼체크가 하나라도 실패하면 return false로
성공하면 return true로 submit이 작동하도록 함

즉, 이벤트를 중지하려면 return false, event.preventDefault() 등의 코드 사용

비밀번호(pwd)
비밀번호 확인(repwd)
이름(name)
성별(gender) 남 여
